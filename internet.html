Clients and Servers

Computers connetced to theWeb are called clients and servers.

  • Clients are the typical Web user's Internet-connected devices (for example, your computer conncted on thos devices (usually a web browser like Firefox or Chrome).
  • Servers are computers that store webpage, sites, or apps.When a client device wants to access a webpage, a copy of the webpage is downloaded from the server onto the client machine to be displayed in the user's web broswer.

So What Actually Happens?

  1. The broswer goes to the DNS server and finds the real address of the Server that the website lives on.
  2. The browser sends an HTTP requestmessage ro the server asking it to send a copy of the website to the client.
  3. If the serever approves the client's request, the server sends the client a "200 OK" message,which means "Permission granted!Here you go!", and then starts sending the website's files to the broswer as a series of small chunks.
  4. The browser assembles the small chunks into a complete website and displays it to you!
